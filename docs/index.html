Installation

1

Clone or Copy Configuration

Terminal 
# For Linux/macOS
cp -r . ~/.config/nvim/

# Or if you want to backup your existing config
mv ~/.config/nvim ~/.config/nvim.backup
cp -r . ~/.config/nvim/
2

Start Neovim

Terminal 
nvim
3

Setup Supermaven (Optional)

When you first start Neovim, Supermaven will prompt you to either:

  • Use the Free Tier: :SupermavenUseFree
  • Activate Pro subscription by following the provided link

Conflict-Free Keybinding System

Zero Keybinding Conflicts

The configuration has been thoroughly analyzed and optimized to ensure all plugins work harmoniously without any keybinding overlaps.

Tab Key Management

Intelligent fallback system between completion, AI suggestions, and syntax navigation. Each plugin gets priority based on context.

Leader Key Organization

Each functionality group has distinct prefixes to prevent overlaps. Navigation, LSP, plugins, and utilities all have their own namespace.

Mode-Specific Mappings

Terminal, insert, and normal mode mappings are properly separated, ensuring no conflicts between different editing modes.

Plugin Harmony

All plugins work together without interfering with each other. LSP, completion, AI suggestions, and file management all coexist peacefully.

Customization

Adding Your Own Settings

Add your custom settings at the end of init.lua:

init.lua 
-- Load core configuration
require("core").setup()

-- Your custom settings here
vim.opt.colorcolumn = "80"
vim.opt.spell = true
vim.opt.spelllang = "en_us"

print("Neovim configuration loaded!")

File Type Specific Settings

Edit lua/core/autocmds.lua to add file type specific settings:

lua/core/autocmds.lua 
-- File type specific settings
vim.api.nvim_create_autocmd("FileType", {
  pattern = { "markdown", "text", "python", "javascript" },
  callback = function()
    if vim.bo.filetype == "markdown" or vim.bo.filetype == "text" then
      vim.opt_local.wrap = true
      vim.opt_local.linebreak = true
      vim.opt_local.textwidth = 80
    elseif vim.bo.filetype == "python" or vim.bo.filetype == "javascript" then
      vim.opt_local.tabstop = 4
      vim.opt_local.shiftwidth = 4
    end
  end
})

Adding New Plugins

Add new plugins to lua/plugins/init.lua:

lua/plugins/init.lua 
M.plugins = {
  -- Your existing plugins...
  
  -- New plugin example
  {
    "username/plugin-name",
    event = "BufRead",
    config = function()
      require("plugin-name").setup({
        -- Plugin configuration
      })
    end,
  },
}

Troubleshooting

Slow Startup

  • Check if you have other Neovim configurations in ~/.config/nvim/
  • Ensure no other init files are being loaded

Plugin Issues

  • Run :Lazy sync to install/update plugins
  • Check :Lazy log for error messages
  • Ensure all dependencies are installed

Supermaven Issues

  • Run :SupermavenShowLog to view logs
  • Check your internet connection
  • Verify your Supermaven account is active
  • Try :SupermavenRestart to restart the service
